Polysonic

HTML5 Subsonic client

Download this project as a .zip file Download this project as a tar.gz file

PolySonic is a HTML5 client for Subsonic.

features

  • modern design HTML5 single page application
  • HTML5 Audio
  • stores art and metadata in indexeddb reducing number of network request and over all improving speed of the app
  • download manager to albums and files to your local machine
  • podcast management
  • playlist management
  • shuffle mode
  • similar songs to album feature
  • sort by folders option
